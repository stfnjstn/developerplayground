## What's New in This Update

**TREASURE CHESTS**
- Add interactive chests to your levels
- Configure chest contents with any game items
- Create engaging treasure hunt experiences
- Optional chests needs a key
- New Trigger events (open, drop, take)

**MONSTER ENHANCEMENTS**
- New monster types with unique abilities
- Complete spell system for monsters
- Advanced monster creation dialog
- Color tinting for monster customization

**EXPANDED GAME ITEMS**
- New 3D models for food and keys
- Additional potion types: permanent skill improvements, haunted spell healing, and more
- Enhanced textures and tinting features
- Improved GameItem creation dialog


**Bug fixes and performance improvements**
