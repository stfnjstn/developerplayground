Treasure hunting comes to "Escape The Dungeon Maze" <span style="background-color: #4CAF50; color: white; padding: 3px 8px; border-radius: 12px; font-size: 0.85em; margin-left: 5px;">R3.0 - December 2025</span> with the addition of the new Chest system!

## New 3D Model: Treasure Chests

Chests are now available in both the Level Editor and the game, opening up exciting new gameplay possibilities:

**Chest Features:**
- **Treasure Storage**: Chests can contain valuable items and rewards
- **Custom Contents**: Place specific items inside chests in the Level Editor
- **Event Triggers**: Chests have an onOpen event that can trigger actions when opened
- **Locked Chests**: Some chests require keys to open - better find that right colored key!

This feature combines beautifully with the key system - dungeon designers can now create quests where players must find specific keys to unlock treasure chests. The onOpen event system also allows for creative puzzles where opening a chest triggers traps, spawns monsters, or unlocks new areas!

As we approach the December 2025 release, all these features are coming together to create a much richer dungeon crawling experience. Exciting times ahead!
